Welcome to the C-Server Project

This site is powered by a fully custom POSIX-compliant C11 web server using epoll for high-performance I/O, written from scratch for educational and practical purposes.

🔧 Features

  • Fully written in pure C11 — no frameworks, no bloat
  • Supports epoll, non-blocking sockets, and persistent HTTP/1.1 connections
  • Modular routing: serve static files, HTML views, and dynamic JSON APIs
  • Client-side navigation via boot.js and dynamic content loading
  • Live data rendered with Chart.js (for /expenses, etc.)
  • Minimal, custom dark theme styled via CSS variables

📜 Project Structure

/app        → C source code (core, router, handlers)
/var/www    → Static assets (HTML, JS, CSS)
/var/lib    → private data (expenses, drive files)

🛰️ Server Status

Server is running on epoll-based listener, supporting both IPv4 and IPv6.

All responses include proper Content-Length, MIME type detection, and persistent connections when allowed.
